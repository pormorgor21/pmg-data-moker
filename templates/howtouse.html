Data Mocker

  • How to Use
  • 1. Enter Number of Column.
  • 2. Enter Name and Value to each columns.
  • 3. Enter Total Rows.
  • 4. Click Let's Mock!.
  • 5. Copy Data to your Excel and Let's fun with your Data.

  • Warning
  • Using comma(,) for separate any Values.
  • Please avoid using " and ' in Value Field .
