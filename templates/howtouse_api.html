Data Mocker

  • How to Use
  • End Point: /mock
  • Method: POST
  • Body: JSON Format
  • Required Field :
  • "ColumnName": ["Value1","Value2","Value3"],
  • "rowNo": 200
  • Warning
  • Avoid using " "(space) in Define Column Name Step.
  • Value in each Columns must be in [ ] (Blacket Close)
  • Using comma(,) for separate any Values.
  • Please avoid using " and ' in Value Field .
